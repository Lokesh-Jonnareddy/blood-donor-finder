# 🩸 Blood Donor Finder System

A web application that connects blood donors with recipients 
based on blood group and location in Chennai.

## 🌟 Unique Features
- 🔴 Urgency Level System (Critical / Normal / Planned)
- 🏆 Donor Badge & Reward System (New Donor → Hero Donor)
- 📍 Area-based donor matching (Chennai zones)
- ⏳ 90-day donation gap tracker
- 👤 Admin dashboard to manage donors & requests

## 💻 Tech Stack
- Backend  : Java, Servlets, JDBC
- Frontend : JSP, HTML, CSS
- Database : PostgreSQL
- Server   : Apache Tomcat
- IDE      : Eclipse

## 🚀 How to Run
1. Clone this repository
2. Import into Eclipse as Dynamic Web Project
3. Set up PostgreSQL database
4. Update DBConnection.java with your credentials
5. Deploy on Apache Tomcat
6. Open http://localhost:8080/BloodDonorFinder
